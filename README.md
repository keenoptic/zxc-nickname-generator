## 🔮 Как это работает

Генератор использует **алгоритм замены символов**, где каждая русская буква преобразуется в стильный символ:

### 📝 Словарь замен

**Гласные буквы:**
```
а → a    е → e    и → u    о → o    у → y
ы → bl   э → 3    ю → I-O  я → 9l
```

**Согласные буквы:**
```
б → 6    в → B    г → r    д → D    ж → }|{
з → 3    к → K    л → Jl   м → M    н → H
п → TT   р → p    с → C    т → T    ф → 8
х → X    ц → C    ч → 4    ш → W    щ → W]
ъ → b    ь → b
```

**Пробел** → `_` (символ подчеркивания)
